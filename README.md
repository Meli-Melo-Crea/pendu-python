# 🎮 Jeu du Pendu - Python

Jeu du pendu classique codé en Python avec ASCII art et liste de mots Final Fantasy.

## 📋 Fonctionnalités

- Affichage progressif du bonhomme pendu en ASCII art (7 étapes)
- Liste de mots personnalisée Final Fantasy (Arias, Tidus, Azastus, Sacrifissia, Enshrouded, Bahamut)
- Suivi des lettres déjà proposées
- Détection des doublons
- Gestion victoire/défaite
- Input normalisé en majuscules

## 🚀 Comment jouer

1. Lance le script : `python pendu.py`
2. Devine les lettres du mot mystère
3. Tu as 7 essais avant de perdre !

## 🛠️ Technologies

- Python 3
- Module `random`
